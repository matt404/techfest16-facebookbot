## Development Steps

1. Merge updates into master branch
2. SSH to the server and execute the following:
3. `cd ~/techfest16-facebookbot`
4. `git pull`
5. `cd app`
6. `npm start`
7. Now log into facebook and go to the 'gdtestbot' page.
8. Open the messenger and chat away!
